FAQ

1) Apakah Anda menyediakan OpenAI API key? Apa yang perlu saya siapkan?
Anda membawa OpenAI API key Anda sendiri. Kami tidak menjual key dan tidak menambahkan markup pada penggunaan OpenAI.
2) Apakah OpenAI key saya aman?
Key Anda disimpan di mesin khusus Anda (tidak dibagikan dengan pengguna lain). Kami tidak pernah menampilkannya kembali di chat. Anda bisa menggantinya kapan saja.
3) Bisa batal kapan saja? Apakah akan ditagih lagi?
Ya. Anda bisa membatalkan kapan saja di WhatsApp. Setelah dibatalkan, Anda tidak akan ditagih bulan berikutnya.
4) Bisa dipakai di grup?
Tidak secara default. Produk ini dioptimalkan untuk “message yourself” (self‑chat) agar tetap privat dan andal.
5) Bagaimana jika saya tidak sengaja melepas tautan perangkat WhatsApp? (Relink)
Jika Anda sudah membayar dan langganan Anda aktif, Anda bisa relink melalui tautan pribadi di p.bothook.me dengan UUID Anda. Kami akan memverifikasi akun WhatsApp cocok sebelum memulihkan akses.
6) Model apa yang bisa saya gunakan?
Anda bisa menggunakan model OpenAI apa pun yang tersedia untuk key Anda (sesuai batas OpenAI). Default diatur untuk kualitas dan stabilitas.
7) Apa batasannya? (kecepatan / rate / penggunaan)
Batasan terutama bergantung pada akun OpenAI Anda dan model yang dipilih. Resource server dedicated dan selalu aktif.
8) Di mana saya menggunakan asisten?
Di WhatsApp. Cara paling mudah adalah membuka chat “Message yourself” lalu berbicara dengan asisten di sana (self‑chat).
9) Bagaimana cara memulihkan UUID saya?
UUID Anda disimpan di server pada file /opt/bothook/UUID.txt. Anda bisa meminta OpenClaw menampilkannya, atau login via SSH dan jalankan cat /opt/bothook/UUID.txt.
10) Untuk apa tautan UUID saya?
Tautan UUID (https://p.bothook.me/p/<uuid>?lang=<lang>) adalah halaman kontrol Anda. Tampilannya bergantung status: tautkan perangkat (baru), relink/batal/upgrade (langganan aktif), atau berlangganan lagi (kedaluwarsa/dibatalkan).
11) Bolehkah saya membagikan UUID / tautan UUID?
Tidak. Anggap sebagai pintu masuk akun. Jangan posting publik. Jika butuh bantuan, bagikan hanya ke dukungan resmi (support@bothook.me).
