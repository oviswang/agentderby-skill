Kebijakan Privasi

Terakhir diperbarui:

Kebijakan Privasi ini menjelaskan bagaimana BOTHook (“bothook”, “kami”) menangani informasi saat Anda menggunakan situs web dan layanan setup OpenClaw yang di‑host.

1. Data yang kami kumpulkan

  • Data penggunaan situs: log dasar yang diperlukan untuk keamanan dan operasional (mis. timestamp, alamat IP, user agent).
  • Pengiriman formulir kontak: email yang Anda berikan, pesan Anda, dan (opsional) Provisioning UUID.
  • Metadata layanan: data operasional minimal untuk mem‑provision dan mengelola instance (mis. identifier order/sesi, status).

2. Konten chat

Saat Anda berbicara dengan bot di aplikasi chat, isi pesan diproses oleh platform chat terkait dan runtime OpenClaw. Kami tidak menjual konten chat. Kami meminimalkan penyimpanan konten dan hanya menyimpan yang diperlukan untuk operasional, dukungan, dan debugging.

3. Cara kami menggunakan data

  • Untuk mem‑provision dan menjalankan instance OpenClaw Anda.
  • Untuk memberikan dukungan dan menanggapi permintaan Anda.
  • Untuk melindungi layanan dari penyalahgunaan dan meningkatkan keandalan.

4. Berbagi data

Kami dapat membagikan data kepada penyedia layanan hanya sejauh diperlukan untuk mengoperasikan bothook (mis. infrastruktur cloud, pengiriman email). Kami tidak menjual informasi pribadi.

5. Retensi data

Kami menyimpan informasi hanya selama diperlukan untuk menyediakan layanan, memenuhi kewajiban hukum, menyelesaikan sengketa, dan menegakkan perjanjian. Jika paket Anda berakhir dan tidak diperpanjang, bot akan berhenti; data operasional dapat tersimpan untuk waktu terbatas untuk keperluan akuntansi dan dukungan.

6. Keamanan

Kami mengambil langkah yang wajar untuk melindungi informasi (kontrol akses, enkripsi saat transit, dan praktik operasional least‑privilege). Namun, tidak ada sistem yang sepenuhnya aman.

7. Pilihan Anda

  • Anda dapat menghubungi kami untuk meminta akses, koreksi, atau penghapusan informasi kontak yang Anda kirimkan.
  • Anda dapat berhenti menggunakan layanan kapan saja dengan tidak memperpanjang.

8. Kontak

Pertanyaan tentang privasi? Gunakan formulir kontak.
