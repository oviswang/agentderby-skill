Kebijakan Privasi

Terakhir diperbarui:

Kebijakan ini menjelaskan bagaimana BOTHook (“bothook”, “kami”) menangani informasi saat Anda menggunakan situs web dan layanan setup OpenClaw yang di‑host.

1. Data yang kami kumpulkan

  • Data penggunaan situs: log dasar untuk keamanan dan operasional (mis. waktu, IP, user agent).
  • Pengiriman formulir kontak: email yang Anda berikan, pesan Anda, dan (opsional) UUID provisioning.
  • Metadata layanan: data minimal untuk mem‑provision dan mengelola instance (mis. identifier order/sesi, status).

2. Konten chat

Saat Anda berbicara dengan bot di aplikasi chat, isi pesan diproses oleh platform chat terkait dan runtime OpenClaw. Kami tidak menjual konten chat. Kami meminimalkan penyimpanan konten dan hanya menyimpan yang diperlukan untuk operasional, dukungan, dan debugging.

3. Cara kami menggunakan data

  • Untuk mem‑provision dan menjalankan instance OpenClaw Anda.
  • Untuk memberikan dukungan dan menanggapi permintaan Anda.
  • Untuk melindungi layanan dari penyalahgunaan dan meningkatkan keandalan.

4. Berbagi data

Kami tidak menjual data pribadi. Kami dapat membagikan data minimal kepada penyedia infrastruktur untuk menjalankan layanan, atau jika diwajibkan oleh hukum.

5. Retensi

Kami menyimpan data hanya selama diperlukan untuk operasional/dukungan, lalu menghapus atau menganonimkan jika sesuai.

6. Kontak

Jika Anda punya pertanyaan tentang privasi, silakan hubungi melalui halaman Contact.
