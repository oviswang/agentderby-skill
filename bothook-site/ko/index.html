자주 묻는 질문

1) OpenAI API 키를 제공하나요? 무엇을 준비해야 하나요?
OpenAI API 키는 직접 준비하셔야 합니다. 키를 판매하거나 OpenAI 사용량에 추가 요금을 붙이지 않습니다.
2) OpenAI 키는 안전한가요?
키는 사용자 전용 머신에 저장되며(다른 사용자와 공유하지 않음), 채팅에 다시 표시되지 않습니다. 언제든 교체할 수 있습니다.
3) 언제든 취소할 수 있나요? 다음 달도 결제되나요?
네. WhatsApp에서 언제든 구독을 취소할 수 있으며, 취소 후에는 다음 달에 자동 결제되지 않습니다.
4) 그룹 채팅에서 쓸 수 있나요?
기본적으로는 지원하지 않습니다. 프라이버시와 안정성을 위해 “나에게 메시지(셀프 채팅)”에 최적화되어 있습니다.
5) WhatsApp 기기 연결을 실수로 해제했어요. (Relink)
결제 완료 + 구독 유효 기간 내라면 p.bothook.me의 개인 링크(UUID)로 다시 연결할 수 있습니다. 복구 전 계정 일치 여부를 확인합니다.
6) 어떤 모델을 사용할 수 있나요?
사용자 OpenAI 키에서 사용 가능한 모델을 사용할 수 있습니다(OpenAI 제한에 따름). 기본 설정은 품질과 안정성 위주입니다.
7) 제한이 있나요? (속도/빈도/사용량)
제한은 주로 OpenAI 계정과 선택한 모델에 따라 달라집니다. 서버는 전용이며 24×7 상시 실행됩니다.
8) 어디에서 사용하나요?
WhatsApp에서 사용합니다. 가장 쉬운 방법은 “나에게 메시지(Message yourself)” 채팅을 열고 그곳에서 대화하는 것입니다.
9) UUID를 어떻게 다시 찾을 수 있나요?
UUID는 서버의 고정 파일 /opt/bothook/UUID.txt 에 저장됩니다. OpenClaw에게 출력해 달라고 하거나, SSH로 접속해 cat /opt/bothook/UUID.txt 를 실행하세요.
10) UUID 링크는 어디에 쓰나요?
UUID 링크(https://p.bothook.me/p/<uuid>?lang=<lang>)는 상태에 따라 다른 기능을 보여주는 컨트롤 페이지입니다: 신규 사용자는 기기 연결, 구독 활성 사용자는 relink/취소/업그레이드, 만료/취소 사용자는 재구독 안내가 표시됩니다.
11) UUID/UUID 링크를 다른 사람에게 공유해도 되나요?
권장하지 않습니다. 서비스 진입점에 해당하므로 공개적으로 공유하지 마세요. 도움이 필요하면 공식 지원(support@bothook.me)에게만 공유하세요.
