Política de privacidad

Última actualización:

Esta Política de Privacidad explica cómo BOTHook (“bothook”, “nosotros”) gestiona la información cuando utilizas nuestro sitio web y el servicio alojado de configuración de OpenClaw.

1. Datos que recopilamos

  • Datos de uso del sitio: registros básicos necesarios para la seguridad y la operación (p. ej., marcas de tiempo, dirección IP, user agent).
  • Envíos del formulario de contacto: el email que proporcionas, tu mensaje y (opcionalmente) tu UUID de provisioning.
  • Metadatos del servicio: datos operativos mínimos necesarios para provisionar y gestionar tu instancia (p. ej., identificadores de pedido/sesión, estado).

2. Contenido de chat

Cuando hablas con el bot dentro de una app de chat, el contenido de esos mensajes lo gestiona la plataforma de chat correspondiente y el runtime de OpenClaw. No vendemos contenido de chat. Intentamos minimizar el almacenamiento del contenido y conservar solo lo necesario para la operación del servicio, soporte y depuración.

3. Cómo usamos los datos

  • Para provisionar y operar tu instancia de OpenClaw.
  • Para ofrecer soporte y responder a tus solicitudes.
  • Para proteger el servicio frente a abusos y mejorar la fiabilidad.

4. Compartición de datos

Podemos compartir datos con proveedores de servicios solo en la medida necesaria para operar bothook (p. ej., infraestructura en la nube, entrega de emails). No vendemos información personal.

5. Retención de datos

Conservamos la información solo el tiempo necesario para prestar el servicio, cumplir obligaciones legales, resolver disputas y hacer cumplir nuestros acuerdos. Si tu plan expira y no renuevas, el bot se detendrá; algunos datos operativos pueden mantenerse durante un periodo limitado por motivos de contabilidad y soporte.

6. Seguridad

Tomamos medidas razonables para proteger la información (controles de acceso, cifrado en tránsito y prácticas operativas de mínimo privilegio). Sin embargo, ningún sistema es perfectamente seguro.

7. Tus opciones

  • Puedes contactarnos para solicitar acceso, corrección o eliminación de la información de contacto que hayas enviado.
  • Puedes dejar de usar el servicio en cualquier momento simplemente no renovando.
