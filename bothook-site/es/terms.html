Términos del servicio

Última actualización:

BOTHook (“bothook”, “nosotros”) ofrece una experiencia alojada de configuración y entrega para ejecutar OpenClaw en apps de chat. Estos términos rigen tu uso del sitio web y del servicio.

1. Lo que ofrecemos

  • Provisionamos una instancia de OpenClaw y ayudamos a conectarla a las apps de chat compatibles.
  • El sitio web se usa para empezar, pagar y ver información básica de estado/facturación.
  • OpenClaw es software de terceros / open‑source; bothook no reclama propiedad sobre OpenClaw.

2. Tus responsabilidades

  • Eres responsable del contenido que envías al bot y de cumplir las reglas de las plataformas de chat que uses.
  • No uses el servicio para actividades ilegales, dañinas, abusivas o que infrinjan derechos.
  • Mantén seguro tu dispositivo y las cuentas de chat vinculadas. Si sospechas un compromiso, contáctanos de inmediato.

3. Disponibilidad y cambios

Buscamos un servicio fiable, pero no se garantiza el uptime. Podemos actualizar, modificar o discontinuar partes del servicio para mejorar la estabilidad o la seguridad.

4. Pruebas, suscripciones, renovación, caducidad

El acceso al servicio está limitado en el tiempo según tu plan. Tras la caducidad, el servicio puede suspenderse o recuperarse de acuerdo con la política operativa.

5. Limitación de responsabilidad

En la medida máxima permitida por la ley, bothook no será responsable por daños indirectos o consecuentes. La responsabilidad se limita al importe que hayas pagado por el periodo actual.
