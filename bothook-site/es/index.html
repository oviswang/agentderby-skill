¿Qué es bothook.me?

bothook offers a simpler way to use OpenClaw. It doesn’t change OpenClaw, and it doesn’t ask you to install or maintain anything yourself.

The bot appears directly in the chat apps you already use. All you need to do is start chatting.

Lo que bothook hace realmente

bothook doesn’t create a new bot. And it doesn’t add another complicated layer on top of OpenClaw.

It does one simple thing: it sets up OpenClaw for you and brings it directly into the chat apps you already use.

Cómo se siente usarlo

  • You don’t use a bot inside a website.
  • You don’t need to learn a new interface.
  • Open your chat app as usual.
  • Start a conversation.
  • You’re already using OpenClaw.

You talk to it on WhatsApp — right where you already are. We’re intentionally focused on doing one channel extremely well.

Para quién es (y para quién no)

Para ti si
  • you want to use OpenClaw, not learn how to install it
  • you prefer getting things done in chat apps
  • you care about using the bot, not running it
No es ideal si
  • you want full control over installation and runtime
  • you need deep low-level customization
  • you enjoy setting up, maintaining, and debugging systems

En una frase: bothook is for people who want to use a bot, not for people who want to build one.

Preguntas frecuentes

1) ¿Ustedes proporcionan la clave API de OpenAI? ¿Qué necesito preparar?
Tú traes tu propia clave API de OpenAI. No vendemos claves ni cobramos margen sobre el uso de OpenAI.
2) ¿Mi clave de OpenAI está segura?
Tu clave se guarda en tu máquina dedicada (no compartida). Nunca se muestra de nuevo en el chat. Puedes cambiarla cuando quieras.
3) ¿Puedo cancelar en cualquier momento? ¿Me cobrarán el próximo mes?
Sí. Puedes cancelar en cualquier momento dentro de WhatsApp. Después de cancelar, no se te cobrará el próximo mes.
4) ¿Puedo usarlo en grupos?
No por defecto. El producto está optimizado para “Message yourself” (chat contigo mismo) para mayor privacidad y fiabilidad.
5) ¿Y si desvinculé el dispositivo de WhatsApp por error? (Relink)
Si pagaste y tu suscripción está activa, puedes volver a vincular desde tu enlace personal en p.bothook.me (UUID). Verificaremos que la cuenta de WhatsApp coincida antes de restaurar el acceso.
6) ¿Qué modelos puedo usar?
Puedes usar cualquier modelo disponible para tu clave (según los límites de OpenAI). Los valores por defecto priorizan calidad y estabilidad.
7) ¿Cuáles son los límites? (velocidad / frecuencia / uso)
Dependen sobre todo de tu cuenta de OpenAI y del modelo elegido. Los recursos del servidor son dedicados y siempre activos.
8) ¿Dónde uso el asistente?
En WhatsApp. Lo más fácil es abrir el chat “Message yourself” y hablar con el asistente allí (self‑chat).

Resumen

bothook es una forma más simple de usar OpenClaw. No cambia OpenClaw: solo lo lleva directamente a WhatsApp.
