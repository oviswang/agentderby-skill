bothook.me 是什麼？

bothook offers a simpler way to use OpenClaw. It doesn’t change OpenClaw, and it doesn’t ask you to install or maintain anything yourself.

The bot appears directly in the chat apps you already use. All you need to do is start chatting.

bothook 實際做了什麼

bothook doesn’t create a new bot. And it doesn’t add another complicated layer on top of OpenClaw.

It does one simple thing: it sets up OpenClaw for you and brings it directly into the chat apps you already use.

使用體驗是什麼樣的

  • You don’t use a bot inside a website.

  • Open your chat app as usual.
  • Start a conversation.
  • You’re already using OpenClaw.

You talk to it on WhatsApp — right where you already are. We’re intentionally focused on doing one channel extremely well.

適合誰（以及不適合誰）

常見問題

1）需要我提供 OpenAI Key 嗎？需要準備什麼？
你需要自備 OpenAI API Key。我們不售賣 Key，也不會對 OpenAI 用量加價。
2）我的 OpenAI Key 安全嗎？
你的 Key 會存放在你的獨立機器上（不與其他使用者共享），不會在聊天中回顯。你可以隨時更換。
3）可以隨時取消訂閱嗎？會繼續扣款嗎？
可以。你可以隨時在 WhatsApp 內取消訂閱。取消後，下個月將不會再扣款。
4）可以在群組裡用嗎？
預設不支援。產品優先最佳化「傳訊息給自己/自聊」，更私密也更穩定。
5）如果我不小心取消了 WhatsApp 裝置連結，怎麼找回？（Relink）
如果你已付費且未到期，可以透過 p.bothook.me 的個人連結（含 UUID）重新連結。系統會核對掃碼的 WhatsApp 帳號一致後才恢復，防止被盜用。
6）我可以用哪些模型？
你可以使用你的 OpenAI Key 所支援的模型（受 OpenAI 自身限制）。預設設定優先穩定與效果。
7）有什麼限制嗎？（速度/頻率/用量）
限制主要取決於你的 OpenAI 帳號與模型選擇；伺服器資源為獨立常駐執行環境。
8）我在哪裡使用助手？
在 WhatsApp 裡使用。最簡單的方式是打開「傳訊息給自己/自聊（Message yourself）」的對話，直接在裡面和助手聊天。
