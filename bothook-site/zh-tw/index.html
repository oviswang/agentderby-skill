bothook 實際做了什麼

常見問題

1）需要我提供 OpenAI Key 嗎？需要準備什麼？
你需要自備 OpenAI API Key。我們不售賣 Key，也不會對 OpenAI 用量加價。
2）我的 OpenAI Key 安全嗎？
你的 Key 會存放在你的獨立機器上（不與其他使用者共享），不會在聊天中回顯。你可以隨時更換。
3）可以隨時取消訂閱嗎？會繼續扣款嗎？
可以。你可以隨時在 WhatsApp 內取消訂閱。取消後，下個月將不會再扣款。
4）可以在群組裡用嗎？
預設不支援。產品優先最佳化「傳訊息給自己/自聊」，更私密也更穩定。
5）如果我不小心取消了 WhatsApp 裝置連結，怎麼找回？（Relink）
如果你已付費且未到期，可以透過 p.bothook.me 的個人連結（含 UUID）重新連結。系統會核對掃碼的 WhatsApp 帳號一致後才恢復，防止被盜用。
6）我可以用哪些模型？
你可以使用你的 OpenAI Key 所支援的模型（受 OpenAI 自身限制）。預設設定優先穩定與效果。
7）有什麼限制嗎？（速度/頻率/用量）
限制主要取決於你的 OpenAI 帳號與模型選擇；伺服器資源為獨立常駐執行環境。
8）我在哪裡使用助手？
在 WhatsApp 裡使用。最簡單的方式是打開「傳訊息給自己/自聊（Message yourself）」的對話，直接在裡面和助手聊天。
9) 如何找回我的 UUID？
你的 UUID 會保存在伺服器本機固定檔案：/opt/bothook/UUID.txt。你可以請 OpenClaw 幫你輸出該檔案內容；或用 SSH 登入後執行：cat /opt/bothook/UUID.txt。
10) UUID 連結有什麼用？
UUID 連結（https://p.bothook.me/p/<uuid>?lang=<lang>）是你的控制入口，會依狀態顯示不同內容：新用戶掃碼綁定；已付費未到期提供 Relink/取消訂閱/升級；已到期或已取消引導重新訂閱。
11) UUID/UUID 連結可以分享給別人嗎？
不建議。它相當於你的服務入口。請勿公開張貼。如需協助，請只透過官方支援（support@bothook.me）。
