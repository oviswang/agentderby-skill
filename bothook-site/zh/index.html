bothook.me 是什么？

bothook 提供一种更简单的方式来使用 OpenClaw：它不会改变 OpenClaw，也不要求你自己安装、维护任何东西。

机器人会直接出现在你已经在用的聊天软件里。你只需要开始聊天。

bothook 实际做了什么

bothook 不会创造一个新机器人，也不会在 OpenClaw 上再叠一层复杂系统。

它只做一件事：替你把 OpenClaw 配好，并交付到你的聊天软件里。

常见问题

1）需要我提供 OpenAI Key 吗？需要准备什么？
你需要自备 OpenAI API Key。我们不售卖 Key，也不会对 OpenAI 用量加价。
2）我的 OpenAI Key 安全吗？
你的 Key 会存放在你的独立机器上（不与其他用户共享），不会在聊天中回显。你可以随时更换。
3）可以随时取消订阅吗？会继续扣费吗？
可以。你可以随时在 WhatsApp 内取消订阅。取消后，下个月将不会再扣费。
4）可以在群里用吗？
默认不支持。产品优先优化“给自己发消息/自聊”，更私密也更稳定。
5）如果我不小心取消了 WhatsApp 设备关联，怎么找回？（Relink）
如果你已付费且未到期，可以通过 p.bothook.me 的个人链接（带 UUID）重新关联。系统会校验扫码账号一致后才恢复，防止被盗用。
6）我可以用哪些模型？
你可以使用你的 OpenAI Key 所支持的模型（受 OpenAI 自身限制）。默认配置优先稳定和效果。
7）有什么限制吗？（速度/频率/用量）
限制主要取决于你的 OpenAI 账号与模型选择；服务器资源为独立常驻运行环境。
8）我在哪里使用助手？
在 WhatsApp 里使用。最简单的方式是打开“给自己发消息/自聊（Message yourself）”这个对话，然后直接在里面和助手聊天。
9) 如何找回我的 UUID？
你的 UUID 会保存在服务器本地固定文件：/opt/bothook/UUID.txt。你可以让 OpenClaw 帮你打印该文件内容；或用 SSH 登录后执行：cat /opt/bothook/UUID.txt。
10) UUID 链接有什么用？
UUID 链接（https://p.bothook.me/p/<uuid>?lang=<lang>）是你的控制入口，会根据状态显示不同内容：新用户扫码关联设备；已付费未到期提供 Relink/取消订阅/升级套餐；已到期或已取消引导重新订阅。
11) UUID/UUID 链接可以分享给别人吗？
不建议。它相当于你的服务入口。请勿公开发布。如需协助，请只通过官方支持渠道（support@bothook.me）沟通。

总结

bothook 是一种更简单的 OpenClaw 使用方式。它不改变 OpenClaw 的工作方式——只是把它更直接地交付给你。

你不需要理解它如何运行，你只需要开始聊天。
