FAQ

1) क्या आप OpenAI API key देते हैं? मुझे क्या तैयार करना है?
आप अपना OpenAI API key खुद लाते हैं। हम keys नहीं बेचते और OpenAI usage पर कोई markup नहीं लगाते।
2) क्या मेरा OpenAI key सुरक्षित है?
आपका key आपकी dedicated machine पर स्टोर होता है (दूसरे users के साथ साझा नहीं)। हम इसे chat में वापस नहीं दिखाते। आप इसे कभी भी बदल सकते हैं।
3) क्या मैं कभी भी cancel कर सकता हूँ? क्या फिर से charge होगा?
हाँ। आप WhatsApp के अंदर कभी भी cancel कर सकते हैं। cancel करने के बाद अगले महीने charge नहीं होगा।
4) क्या मैं इसे groups में इस्तेमाल कर सकता हूँ?
डिफ़ॉल्ट रूप से नहीं। यह प्रोडक्ट “message yourself” (self‑chat) के लिए optimize किया गया है ताकि यह private और reliable रहे।
5) अगर मैंने गलती से WhatsApp device unlink कर दिया तो? (Relink)
अगर आपने भुगतान किया है और subscription active है, तो आप p.bothook.me पर अपने personal link (UUID) से relink कर सकते हैं। access restore करने से पहले हम WhatsApp account match verify करेंगे।
6) मैं कौन‑से models इस्तेमाल कर सकता हूँ?
आप अपने key के लिए उपलब्ध कोई भी OpenAI model इस्तेमाल कर सकते हैं (OpenAI limits के अनुसार)। defaults quality और stability के लिए tuned हैं।
7) limits क्या हैं? (speed / rate / usage)
limits मुख्यतः आपके OpenAI account और चुने गए model पर निर्भर हैं। server resources dedicated हैं और हमेशा on रहते हैं।
8) मैं assistant कहाँ उपयोग करूँ?
WhatsApp में। सबसे आसान तरीका है “Message yourself” chat खोलना और वहीं assistant से बात करना (self‑chat)।
9) मैं अपना UUID कैसे वापस पा सकता/सकती हूँ?
आपका UUID आपके सर्वर पर /opt/bothook/UUID.txt फ़ाइल में सेव होता है। आप OpenClaw से इसे दिखाने को कह सकते हैं, या SSH से लॉग‑इन करके cat /opt/bothook/UUID.txt चलाएँ।
10) UUID लिंक किस काम आता है?
UUID लिंक (https://p.bothook.me/p/<uuid>?lang=<lang>) आपका कंट्रोल पेज है। यह स्टेटस के अनुसार अलग विकल्प दिखाता है: नया यूज़र—डिवाइस लिंक; एक्टिव सब्सक्रिप्शन—relink/कैंसल/अपग्रेड; एक्सपायर्ड/कैंसल—फिर से सब्सक्राइब।
11) क्या मैं अपना UUID / UUID लिंक किसी के साथ शेयर कर सकता/सकती हूँ?
नहीं। इसे अकाउंट एंट्री‑पॉइंट मानें। सार्वजनिक रूप से शेयर न करें। मदद चाहिए तो केवल आधिकारिक सपोर्ट (support@bothook.me) के साथ शेयर करें।
